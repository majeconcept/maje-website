**1. globals.css — CSS guard pointer:coarse (per CURS-06, RESP-01):**

Ajouter à la fin de `src/app/globals.css` le bloc media query suivant. Le curseur JS bails out sur pointer:coarse déjà, mais cette règle CSS garantit que les éléments sont cachés AVANT le chargement du JS (évite un flash du curseur sur touch):

```css
/* ── Touch/mobile: hide custom cursor elements before JS loads ── */
@media (pointer: coarse) {
  .cursor-dot,
  .cursor-ring,
  .cursor-follower,
  .cursor-spotlight {
    display: none !important;
  }

  * {
    cursor: auto !important;
  }
}
```
