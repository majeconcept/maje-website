Créer deux nouveaux fichiers dans `src/app/` — Next.js les transforme automatiquement en routes `/sitemap.xml` et `/robots.txt`.

**src/app/sitemap.ts** (per SEO-03) :
```typescript
import type { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  const baseUrl = process.env.NEXT_PUBLIC_SITE_URL ?? 'https://maje-concept.fr'

  return [
    {
      url: baseUrl,
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 1,
    },
  ]
}
```

Site single-page — une seule entrée `/`. Les sections (#portfolio, #contact, etc.) sont des ancres dans la homepage, pas des routes séparées.

**src/app/robots.ts** (per SEO-03) :
```typescript
import type { MetadataRoute } from 'next'

export default function robots(): MetadataRoute.Robots {
  const baseUrl = process.env.NEXT_PUBLIC_SITE_URL ?? 'https://maje-concept.fr'

  return {
    rules: [
      {
        userAgent: '*',
        allow: '/',
        disallow: ['/api/'],
      },
    ],
    sitemap: `${baseUrl}/sitemap.xml`,
  }
}
```

`/api/` disallowed — le route handler `/api/contact` ne doit pas être indexé.
